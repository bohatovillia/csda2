# CSDA Проєкт

## Опис проєкту
Цей репозиторій містить набір лабораторних робіт з дисципліни "Сучасні технології розробки програмного забезпечення для вбудованих систем" (CSDA2). Усі лабораторні роботи спрямовані на розробку програмного забезпечення для мікроконтролера PSoC 6 (CY8CKIT-062-WIFI-BT) із застосуванням сучасних веб-технологій та мережевих протоколів.

## Структура репозиторію (Перелік лабораторних робіт)

* **[Lab 1: Розробка вбудованого вебсервера](./lab1/)**
  Реалізація простого HTTP-сервера на базі мікроконтролера, налаштування модулю Wi-Fi в режимі Station та інтеграція файлової системи LittleFS для зберігання і роздачі статичної вебсторінки `index.html`.

* **[Lab 2: Асинхронне керування периферією](./lab2/)**
  Створення RESTful API. Додавання обробки `GET` та `POST` HTTP запитів у форматі JSON (за допомогою бібліотеки cJSON) для асинхронного керування світлодіодом та перехоплення стану периферії без перезавантаження сторінки (Fetch API).

* **[Lab 3: Візуалізація даних у реальному часі](./lab3/)**
  Інтеграція I2C сенсора акселерометра (MPU-6050). Розгортання WebSocket сервера для двосторонньої потокової передачі даних. Використання JavaScript бібліотеки `Chart.js` для малювання графіків показників сенсора в реальному часі на клієнті.

* **[Lab 4: Конфігурування мереж (Captive Portal)](./lab4/)**
  Переведення пристрою в режим SoftAP. Реалізація DNS Spoofing (UDP:53) для примусового перенаправлення (`192.168.1.1`) пристроїв на свою сторінку. Створення сторінки конфігурації (з Wi-Fi сканером) та збереження SSID/Пароля у незалежній Flash-пам'яті (Emulated EEPROM) з подальшим перезавантаженням (`NVIC_SystemReset`).

* **[Lab 5: Безпека вбудованих вебресурсів](./lab5/)**
  Захист з'єднання і передачі даних. Інтеграція криптографічної бібліотеки `mbedTLS` для роботи HTTPS сервера на порту 443 із самопідписаними RSA сертифікатами. Впровадження HTTP Basic Authentication для обмеження доступу стороннім до панелі керування пристроєм.

## Інструкція із запуску

1. Склонуйте репозиторій:
   ```bash
   git clone https://github.com/bohatovillia/csda2.git
   ```
2. Відкрийте потрібну лабораторну роботу (наприклад `lab5/`) у вашому середовищі розробки, наприклад **ModusToolbox** або **VS Code** із плагінами для Cypress/Infineon.
3. Скомпілюйте прошивку (`main.c`) під таргет вашої плати (за замовчуванням `CY8CKIT-062-WIFI-BT`).
4. Для перевірки фронтенду (де це можливо, наприклад у `lab4` чи `lab5`), можна просто відкрити HTML файли (`demo.html` або `index.html`) у браузері або за допомогою локального севера `npx serve`.

## Використання
Після прошивки мікроконтролера відкрийте Serial Monitor (Baud Rate 115200) для перегляду логів, які повідомлять IP-адресу пристрою, і перейдіть за цією адресою в браузері.
